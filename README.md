# bernyflix

Project created for learning purposes, replicating the visual style of Netflix.

## Dependencies
VueJS
Pinia
Vue-Material-Design-Icons
Vue3-Carousel

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```
